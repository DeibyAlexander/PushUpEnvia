# PushUpEnvia


Para inicializar nuestro backend primero ingresamos a la carpeta "backend"

        cd backend

Ahora instalamos las dependencias necesarias para que funcione correctamente

        npm i

Y ahora podemos podemons correr el servidor

        npm run dev
